# Git 정리

## 명령문

* git status 상태
* git add . 전부 추가
* git init 파일기본 셋팅
* git log 로그
* git commit -m '코멘트' 사이트 올리기전에 대기상태
* git push origin master 깃 사이트에 올리기
* 로그인 후 아이디 계정 아이디 물어봄
* 다시 비번 입력
